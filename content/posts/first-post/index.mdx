처음으로 Gatsby 블로그 만들고 포스트 올리기 까지 과정

node 버전 업그레이드
npm , git 설치 ( 원래 되어 있었음 )

gatsby 설치
npm install -g gatsby-cli

이후 테마 골라서 설치
npx gatsby new "블로그이름" https://github.com/gatsbyjs/gatsby-starter-blog

게시글 올리기
content>posts 에 들어가서 게시글 폴더 만들기

index.mdx에
